Trello-generated implementation plans (Ball_Bounce)

This folder contains implementation plans generated by the assistant for new Trello cards on the Ball_Bounce board.

File naming
- Each plan file is named: {card.shortLink}-{card.id}.md (e.g., VbIMepxG-6a06e4e02ef7b3640c40c0c4.md)

Plan template
- See template.md for the content layout the assistant will use when creating a plan.

Sync behavior
- The assistant treats presence of a plan file as already-synced. To re-generate, delete the corresponding file or add a `regenerate` label to the Trello card.
